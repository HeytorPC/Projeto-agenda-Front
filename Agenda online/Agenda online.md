# **Agenda online para Tatuador**

Agenda para profissionais que trabalham com hora marcada

---

## **Funcionalidades para usuário:**

* Marcar hora
* Desmarcar hora
* Visualizar horários
* aviso de antecedência (WhatsApp + E-mail)
* escolha do serviço
* Envio de imagem da tatuagem (opcional)



## **Funcionalidades para o Tatuador:**

* Marcar hora
* Desmarcar hora
* Visualizar horários





## **Tecnologias usadas:**

* **HTML5, CSS, Javascript**
* **Java**
